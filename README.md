# House Price Prediction using Linear Regression 🏠

## 📌 Objective
Build a linear regression model to predict house prices.

## 🛠 Tools Used
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn

## 🔍 Steps Performed
1. Data loading and preprocessing
2. Train-test split
3. Model training using Linear Regression
4. Model evaluation (MAE, MSE, R²)
5. Visualization of predictions and residuals

## 📊 Results
- Model evaluated using MAE, MSE, and R² score
- Visualization shows relationship between actual and predicted values
